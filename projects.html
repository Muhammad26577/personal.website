Project 1: How does hardware work?

Our team built NOT, AND, and OR logic gates using transistors and breadboard circuitry at ADA University's lab. The project showcases how basic hardware logic functions are implemented using discrete components and wiring techniques.

Key highlights: transistors (BJTs), breadboard wiring, and hands-on circuit debugging.
